clojure-maven-example by talios

Example maven project using the clojure-maven-plugin

Welcome to the clojure-maven-example wiki! This is a simple example maven project showing how to use the clojure-maven-plugin. The repository has three examples separated by tags: * step1 - Basic pom.xml and a simple test script. Run "mvn test" to run the test. * step2 - The compojure webframeworks repository is included as a git submodule and a simple web application added to the project.
